\# 🤖 AI Software Engineer



An autonomous coding agent built with LangGraph + LangChain + Google Gemini.



\## How it works

1\. 🧠 Planner breaks your request into subtasks

2\. 💻 Coder writes Python files

3\. 🔍 Reviewer critiques the code

4\. 🧪 Test Runner executes and validates

5\. 🔄 Loops back if errors found



\## Setup

```bash

python -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

cp .env.example .env  # add your GOOGLE\_API\_KEY

python graph.py

```



\## Tech Stack

\- LangGraph

\- LangChain

\- Google Gemini 2.5 Flash
